# User Flow и Customer Journey Map (CJM)

## Персона
**ML Engineer / AI Researcher**, задача которого — оценить качество работы небольших локальных моделей (SLM/LLM) на специфическом домене (сложные научные тексты на русском языке).

## User Flow (Сценарий использования)
1. Инженер запускает `article_retriever.py`. Скрипт сам собирает базу PDF и переводит их в Markdown. Может потребоваться ручное удаление неподходящих статей (например, иногда могут попасться статьи на татарском, которые нельзя отсеять кодом)
2. Инженер запускает `dataset_getter.py`. Система стучится в Google API и формирует золотой стандарт (Ground Truth) датасета. Если используется бесплатная версия, нужно вручную заменять API ключи или ставить модель, где квота еще не израсходована (У нас везде использовался лучший на тот момент Gemini 3 Flash Preview)
3. Инженер выбирает модель для теста (например, Qwen 3.5), меняет `MODEL_ID` в скрипте и запускает Inference. Если используется бесплатная версия Google Colab, инференс может занять несколько дней
4. Запускается этап оценки (Judge). Те же предостережения, что и для пункта 2
5. Инженер получает итоговый средний балл и JSON-файл для аналитики.

## Customer Journey Map (Карта пути пользователя)

| Этап | Действие пользователя | Точки контакта (Скрипты) | Боли и барьеры (Pain Points) | Решения в коде (Улучшения) |
| :--- | :--- | :--- | :--- | :--- |
| **Сбор данных** | Запуск парсера статей | `article_retriever.py` | IP могут забанить за DDOS; OAI отдает мусорные теги; прерывание сессии. | Добавлен `time.sleep()`, жесткая проверка `<meta keywords>`, **система возобновления (Resume) по наличию файлов**. |
| **Создание датасета** | Генерация QA-пар | `dataset_getter.py` | LLM выдает невалидный JSON; лимиты API (Rate limits). | Очистка через `.find('[')`, паузы между запросами, промежуточные сохранения (каждые 10 статей). |
| **Тестирование LLM** | Инференс локальной модели | `models_inference.py` (Gen) | Out of Memory (OOM) из-за огромных статей; зацикливание CoT-моделей. | Квантизация 4-bit, обрезка контекста (15-20k символов), Regex очистка тега `<think>`, `repetition_penalty=1.2`. |
| **Оценка (Judge)** | Сравнение ответов | `models_inference.py` (Judge) | Ошибки 503 (Overloaded) и 429 от Gemini API при массовой оценке. | Бесконечный цикл `while True` с блоком `try-except` и экспоненциальной задержкой. |
